# eternalMac Website

Marketing homepage and documentation for `eternalMac`.

## Local Preview

```bash
npm install
npm run dev
```

The dev server binds to `127.0.0.1` by default.

## Production Build

```bash
npm run build
```

Cloudflare Pages settings:

- Root directory: `website`
- Build command: `npm run build`
- Build output directory: `dist`
